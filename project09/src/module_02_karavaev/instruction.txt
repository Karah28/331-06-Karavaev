ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРОГРАММЫ

1. ТРЕБОВАНИЯ К СИСТЕМЕ:
   - Java JDK 17 или выше
   - MySQL Server 8.0 или выше
   - Maven 3.6 или выше
   - Браузер (Chrome, Firefox, Edge)

2. НАСТРОЙКА БАЗЫ ДАННЫХ:
   а) Запустите MySQL Server
   б) Откройте MySQL Command Line Client
   в) Выполните команды:
      mysql -u root -p
      (введите пароль root пользователя)
   г) Выполните скрипты в следующем порядке:
      source [путь_к_файлу]/setup.sql
      source [путь_к_файлу]/init.sql

3. НАСТРОЙКА ПРОГРАММЫ:
   а) Откройте командную строку
   б) Перейдите в корневую директорию проекта
   в) Выполните команду:
      mvn clean install

4. ЗАПУСК ПРОГРАММЫ:
   Способ 1 (через run.bat):
   - Дважды кликните на файл run.bat
   - Дождитесь сообщения о успешном запуске

   Способ 2 (через командную строку):
   - Откройте командную строку
   - Перейдите в корневую директорию проекта
   - Выполните команду:
     mvn spring-boot:run

5. ДОСТУП К ПРОГРАММЕ:
   - Откройте браузер
   - Перейдите по адресу: http://localhost:8080
   - Войдите в систему:
     Логин: admin
     Пароль: admin123

6. ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ:
   а) Ошибка подключения к базе данных:
      - Проверьте, запущен ли MySQL Server
      - Проверьте правильность данных подключения в application.properties
   
   б) Ошибка порта занят:
      - Закройте другие приложения, использующие порт 8080
      - Или измените порт в application.properties

   в) Ошибка Maven:
      - Проверьте установку Maven
      - Выполните команду: mvn clean
      - Повторите установку
