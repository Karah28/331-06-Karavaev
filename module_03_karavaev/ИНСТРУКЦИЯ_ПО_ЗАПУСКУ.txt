ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРИЛОЖЕНИЯ
=====================================

ОПИСАНИЕ ПРОЕКТА:
Система управления партнёрами с расчётом скидок

АВТОР: Караваев Александр

СПОСОБЫ ЗАПУСКА:

1. ЗАПУСК ИЗ ИСХОДНОГО КОДА:
   - Откройте командную строку или терминал
   - Перейдите в папку проекта: cd "A:\Загрузки\project_10\module_03_karavaev"
   - Запустите команду: python main.py

2. ЗАПУСК ГОТОВОГО ПРИЛОЖЕНИЯ:
   - Перейдите в папку: dist\PartnerManagement
   - Запустите файл: PartnerManagement.exe

3. СБОРКА ПРИЛОЖЕНИЯ (если нужно пересобрать):
   - В командной строке выполните: pyinstaller PartnerManagement.spec

ТРЕБОВАНИЯ:
- Python 3.7 или выше
- Установленные библиотеки: tkinter, sqlite3

СТРУКТУРА ПРОЕКТА:
- main.py - главный файл приложения
- database.py - работа с базой данных
- discount_calculator.py - расчёт скидок
- styles.py - стили интерфейса
- ui_utils.py - утилиты интерфейса
- partner_system.db - база данных партнёров
- images/ - папка с изображениями
- src/ - исходные файлы
- ui/ - файлы интерфейса
- dist/ - готовое приложение
